# Day 001 - Project North Star

## Decisions
- North Star is the project headquarters.
- Focus on building an Enterprise AI engineering brand, not a job-seeking portfolio.
- EKOS is the flagship platform.
- EKOS should be positioned as enterprise intelligence infrastructure, not as an ontology demo.
- Important strategy discussions must become repository artifacts, not only chat history.

## Key Insights
- Start from enterprise problems, not technology.
- Ontology is valuable because enterprise AI needs explicit semantics and reviewable meaning.
- MCP is an execution interface, not just a connector.
- Every design decision must answer 'Why?'
- AI models change quickly; enterprise semantics change slowly.
- The public narrative should be problem -> enterprise meaning -> EKOS -> implementation layers.
